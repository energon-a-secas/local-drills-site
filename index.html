Quick Start

Prerequisites & setup

# Clone the repository
git clone https://github.com/energon-a-secas/local-drills.git
cd local-drills

# Check requirements (Docker, AWS CLI, jq, curl)
make check-requirements

# AWS drills — start LocalStack
export LOCALSTACK_AUTH_TOKEN=<your-token>
cd aws && make run

# Kubernetes drills — start Minikube
cd kubernetes && make start

# GitLab CI drills — start gitlab-ci-local
cd gitlab && make start
What you get
  • Each drill has a broken environment ready to debug
  • Validation commands tell you when you've fixed it
  • Detailed solutions with root-cause explanations
  • Three difficulty tiers: beginner → advanced
  • Knowledge-check quizzes, no infrastructure needed
  • Scaffold tool to create your own drills
  • Works on Apple Silicon (arm64)
  • Community LocalStack tier for most AWS drills
